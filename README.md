# Praticando SDL fazendo uma implementação genérica do jogo da cobrinha

Trabalho em progresso


## Instalando dependências no Ubuntu

```bash
# lib sdl de dev
sudo apt-get install -y libsdl2-dev
# extensão de TTF
sudo apt-get install -y libsdl2-ttf-dev
```
